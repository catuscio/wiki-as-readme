# Security Policy

## Supported Versions

We only support the latest release of **Wiki As Readme**. Please ensure you are using the most recent version before reporting a vulnerability.

| Version | Supported          |
| ------- | ------------------ |
| Latest  | :white_check_mark: |
| Older   | :x:                |

## Reporting a Vulnerability

We take the security of **Wiki As Readme** seriously. If you have discovered a security vulnerability, please do **not** report it through public GitHub issues or pull requests.

Instead, please report it via email to:

**catuscio@hotmail.com**

Please include as much detail as possible to help us reproduce and fix the issue. We aim to acknowledge your report within 48 hours and will keep you updated on the progress of the fix.
